Creators Steps

To became a creator, an individual must choose to create a project

  • 1

    Create an account and sign in

  • 2

    Click Start your Project

  • 3

    Fill out the application

  • Wait for Approval

  • 4

    If approved, the project will go online for the alloted time period.

  • If project is not approved, the Creator may be asked to sort out some issues within the project.

  • After the allotted period ends the creator would have

  • 5

  • Reached their Project Goal or surpassed it.
  • A successfully supported project

  • Not reached their Project Goal.
  • An unsuccessfully supported project
